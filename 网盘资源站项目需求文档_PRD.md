# 网盘资源站项目需求文档（PRD）

> **目标**：开发一个现代化、高颜值、响应式的网盘资源站，用于展示各类资源，通过夸克网盘、百度网盘、迅雷网盘等网盘链接进行资源分发，实现网盘拉新。

## 一、项目定位

-   高颜值、现代化、响应式
-   GitHub + Cloudflare Pages 部署
-   无数据库，Decap CMS 后台
-   支持新增、编辑、删除资源

## 二、技术栈

-   Astro
-   Tailwind CSS
-   TypeScript
-   Decap CMS
-   Markdown
-   GitHub
-   Cloudflare Pages
-   Pagefind

## 三、设计理念

-   Claude 自由发挥 UI 与配色
-   风格参考 Apple、Linear、Notion、Vercel
-   简洁、留白、现代、舒适
-   不使用博客/下载站模板风格
-   深浅色模式
-   响应式设计
-   高性能
